# 50音学习

## JapaneseStudy-ios上学习日语50音的软件

### 摘要

* 方便学习日语50音.

* 可以便捷查找每个50音，解释和发音

* 复习学过的，对应的辨析，做题测试

* 历史了解50音起源内容，帮助你更好的理解
